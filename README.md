# AlexaApp Tierquiz

Das Repository enthält den Quellcode zur Alexa App "Tierquiz". Hier die Beschreibung aus dem Alexa Skill Store (https://www.amazon.de/dp/B06XK963MZ):

Teste dein Wissen über Tiere.
In diesem Quiz musst du herausfinden welche Eigenschaft zu einem Tier gehört. 
Alexa stellt eine Frage die nur zu einem der Tiere passt und nennt dir 4 Antwortmöglichkeiten. Finde heraus auf welches Tier diese Eigenschaft zutrifft.

Alexa fragt dich zum Beispiel:
Welches der Tiere lebt auf dem Bauernhof? Möwe, Schwein, Eisbär oder Zebra

Deine Antwort wäre dann:
Schwein

Alle verfügbaren Tiere mit den jewiligen Eigenschaften sind hier aufgelistet: https://docs.google.com/spreadsheets/d/e/2PACX-1vQCPMyfTL207BGZq7Su5-oZ3JngjPiFlWpxFeyPd__RER75hAXLUIZHzH1m7DaE5UiwTc7D78tKp6oB/pub?gid=0&single=true&output=pdf
